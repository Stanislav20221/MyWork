Академия ДПО
Cтажировка I Кус Н. Н. Разработка прототипа нейро-координатора для ООО «Академия ДПО»

Проект: Разработка нейро-координатора на базе ChatGPT API для ООО «Академия ДПО». Заказчик: ООО «Академия ДПО» (Академия дополнительного профессионального образования), https://academydpo.org/

Описание проекта: Разработка интеллектуального ассистента (нейро-координатора) для сотрудников и клиентов компании «Академия ДПО». Система будет использовать API ChatGPT для обработки запросов на естественном языке и предоставлять информацию на основе структурированной базы знаний компании. Ассистент предназначен для ответов на вопросы о рабочем процессе, образовательных продуктах (курсах), условиях обучения, внутренних регламентах, а также для помощи клиентам в выборе программ и навигации по услугам. Интеграция планируется в веб-чат на сайте academydpo.org и, возможно, в Telegram-бот.

Цель проекта: Создание прототипа чат-ассистента на базе ChatGPT API для автоматизации и улучшения процессов консультирования сотрудников и клиентов «Академии ДПО», предоставляя быстрый и стандартизированный доступ к актуальной информации из базы знаний компании.
